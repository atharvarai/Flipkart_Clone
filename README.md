# Flipkart_Clone
Flipkart Clone made using MERN stack , Material-UI, Redux and integrated Paytm payment gateway.

The Flipkart Clone project features an existing user login and new user sign-in module, allowing users to create an account and access shopping features like viweing details of products, and saving them to the cart and removing them. It also has a Paytm payment gateway for secure and reliable transactions.
